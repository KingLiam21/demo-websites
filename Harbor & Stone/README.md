# Harbor & Stone

Boutique hotel website for a fictional coastal Maine retreat. The vibe is "quiet luxury" - the kind of place where you forget what day it is.

## Design

Editorial, image-heavy, lots of breathing room. Cormorant Garamond for headings gives it that timeless hotel feel. Earthy palette - stone, sand, driftwood, ocean.

## Features

- Full-screen hero with parallax
- Room cards with photo gallery lightbox
- Comparison table for room amenities
- Guest testimonials
- Weather widget (simulated)
- Curated stay packages (romantic, wellness, writer's retreat)
- Instagram feed in footer
- Newsletter signup
- Loading screen
- Cursor glow effect on desktop
- Mobile hamburger menu
